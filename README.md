# Memory Match — لعبة الذاكرة (Phaser 3)

هذه لعبة ويب بسيطة مبنية باستخدام Phaser 3. تم تصميمها للعمل فورًا بدون أي ملفات وسائط خارجية — كل الخامات مخلوقة ديناميكيًا بالـ canvas، والتأثيرات الصوتية تُولَّد برمجيا باستخدام WebAudio.

محتويات المشروع:
- index.html — صفحة الويب الرئيسية
- src/styles.css — أنماط الصفحة
- src/main.js — منطق اللعبة (Phaser Scenes + WebAudio)
- package.json — سكربتات مساعدة لتشغيل محليًا

تشغيل محلي بسرعة:
1. انسخ المشروع إلى مجلد جديد على جهازك.
2. افتح الطرفية داخل المجلد وشغّل سيرفر بسيط:
   - باستخدام Python:
     ```
     python -m http.server 8000
     ```
     ثم افتح: http://localhost:8000
   - أو باستخدام npx:
     ```
     npm install -g serve   # اختياري
     npx serve .
     ```
3. افتح المتصفح وادخل إلى العنوان، أو افتح index.html (يفضل تشغيل عبر سيرفر لتفادي سياسات المتصفح).

نشر سريع على GitHub Pages:
1. أنشئ مستودعاً جديداً على GitHub باسم مثلا `memory-match-phaser`.
2. ادفع الملفات إلى الفرع `main`.
3. يمكنك إضافة ملف workflow لنشر تلقائي (مثال داخل .github/workflows/pages.yml) أو تفعيل GitHub Pages يدوياً في إعدادات المستودع.
4. بعد النشر ستكون اللعبة متاحة عبر: https://<your-username>.github.io/<repo-name>/.

ميزات حالية:
- لوحة لعبة 4x4 (8 أزواج).
- قائمة رئيسية وسجل أفضل نتيجة محفوظ في localStorage.
- تأثيرات صوتية بسيطة باستخدام WebAudio لا تتطلب ملفات صوت.
- دعم اللمس والماوس، وتصغير/تكبير تلقائي لواجهات الموبايل.
